# Privacy Policy for TokTik Auto

Last Updated: January 9, 2026

## 1. Data Collection

TokTik Auto does not collect, store, or transmit any user data to third-party servers. All processing, including video editing and caption generation via Google AI, happens directly on the operator's local machine or via direct, secure API calls to authorized providers (Google AI and TikTok).

## 2. API Keys and Credentials

API keys (Google AI, TikTok Client Key/Secret) and OAuth tokens are stored locally in the environment variables and configuration files on the user's system. They are never shared or uploaded to any external service.

## 3. Data Usage

The application uses the TikTok API solely for the purpose of uploading drone videos and managing content as directed by the operator.

## 4. Third-Party Services

This tool interacts with:

- **Google AI (Gemini)**: For generating content ideas and scripts.
- **TikTok API**: For content publishing and management.

Users should refer to the privacy policies of these respective providers.

## 5. Contact

For any questions regarding this internal policy, please contact the repository owner.
